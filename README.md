# 律所实习生存模拟器 · HJC の噩梦

一款黑色幽默的**实习劝退互动小游戏**。你是一名满怀憧憬的法学院实习生，即将亲历带教律师 HJC 的**八大名场面**——无限工作量、听不懂人话、"来解决问题不是来学习"、克扣日薪、拖欠报销、砍价修罗场……

**在精神值归零之前，看你能撑几天？**

## 🎮 在线体验

打开：https://Udgtsz.github.io/hjc-sim/

## 🎯 玩法

- 每天面对 HJC 的经典操作，做出你的选择
- 三个核心指标：**精神值 / 钱包 / 天数**
- 五种结局：潇洒逃离 / 天真留守 / 留证维权 / 精神崩溃 / 熬满全程
- 结算单：应得 vs 到手 vs 被薅走
- 玩完可一键复制战绩转发

## 🛠️ 技术

- 纯静态网页（单文件 `index.html`）
- Tailwind CSS + 原生 JS
- 无后端、无追踪、无登录
- 所有数据仅在你的浏览器本地运行
